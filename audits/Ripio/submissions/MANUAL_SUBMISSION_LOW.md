## L-01: LatamStable — Unlimited Mint Allows Infinite Token Supply

**Contract:** LatamStable.sol
**Function:** mint()
**Severity:** Low (CVSSv3: 3.7)
**CWE:** CWE-284

### Description
The `LatamStable.mint()` function allows any address with `MINTER_ROLE` to mint an unlimited amount of the stablecoin. While this is by design for the bridge operator, there is no upper bound or supply cap on the total supply.

If a `MINTER_ROLE` address is compromised, there is no circuit breaker or supply limit to prevent infinite minting of the stablecoin.

### Vulnerable Code
```solidity
function mint(address to, uint256 amount) external onlyRole(MINTER_ROLE) {
    _mint(to, amount);
}
```

### Impact
- A compromised `MINTER_ROLE` can mint unlimited tokens
- No supply cap means the token can be inflated to zero value
- No emergency pause mechanism to stop minting in case of compromise

### Recommended Mitigation
1. Implement a maximum supply cap:
```solidity
uint256 public maxSupply;

function mint(address to, uint256 amount) external onlyRole(MINTER_ROLE) {
    require(totalSupply() + amount <= maxSupply, "Exceeds max supply");
    _mint(to, amount);
}
```

2. Add a pausable mechanism that can halt minting in emergencies.
